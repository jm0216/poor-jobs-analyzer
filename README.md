# poor_jobs_analyzer
Analyze poor jobs

# Install

You need Python 3 only :).

# Run

Get jobs from [pyjobs](http://pyjobs.com.br/):
```
python download_pyjobs.py
```

Or run 'fast version':
```
python download_pyjobs_threadpool.py
```

Before get jobs run analyzer:
```
python analyze.py
```

If you need change metrics alter file `metrics.csv`.
